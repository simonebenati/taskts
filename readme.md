# TaskTS

## Prerequisites
- Node.js
- npm
- Browser (Chrome suggested)

## Quick Start

### 1. Setup Backend
From the root directory:
```bash
cd backend
npm install
npm run build
npm start
```

### 2. Setup Frontend
# *note* Open a new terminal
From the root directory:
```bash
cd frontend
npm install
npm run build
npm run preview
```

> [!NOTE]
> The backend requires `key.pem` and `cert.pem` in the `backend` folder for HTTPS.
